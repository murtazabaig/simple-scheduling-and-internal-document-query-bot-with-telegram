# Simple Scheduling and Internal Document Query Bot with Telegram

Advanced n8n automation for Simple Scheduling and Internal Document Query Bot with Telegram.

## Overview
- Category: Internal Wiki, AI RAG
- Complexity: advanced
- Source: n8n workflow template export

## What This Automation Does
Build a smart Telegram bot that schedules events, checks prices, and queries docs using AI with MCP and RAG. Easy setup, Google integrationlearn more!

## Included Files
- `workflow.json`
- `metadata.json`

## Setup
1. Import `workflow.json` into n8n.
2. Configure required credentials for the services used in the workflow nodes.
3. Update any environment variables or static values inside nodes (API keys, URLs, IDs).
4. Run a test execution and then activate the workflow.
